1. **Clone the repository**:

   
 ```bash
   git clone https://github.com/KarimAdel-1/worldwise.git
   cd worldwise
 ```

2. **Install dependencies**:

   
 ```bash
   npm install
 ```

3. **Start the development server**:

   
 ```bash
   npm run dev
 ```

4. **Start the JSON server** (for backend data):

   
 ```bash
   npm run server
 ```

5. **Build the project**:

   
 ```bash
   npm run build
 ```

6. **Preview the production build**:

   
 ```bash
   npm run preview
 ```
